# 🤖 AI Call Agent

An AI-powered voice calling application built with Python, Flask, Google Gemini AI, and Twilio. This project enables automated voice conversations by integrating AI with real-time telephony services.

## 🚀 Features

- AI-powered voice conversations
- Real-time inbound and outbound calling
- Google Gemini AI integration
- Twilio Voice API integration
- Flask backend with REST APIs
- Webhook support
- Secure environment variable configuration

## 🛠️ Tech Stack

- Python
- Flask
- Google Gemini AI
- Twilio Voice API
- Ngrok

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/AI_CALL_Agent.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file with the following variables

```env
GEMINI_API_KEY=your_api_key
TWILIO_ACCOUNT_SID=your_account_sid
TWILIO_AUTH_TOKEN=your_auth_token
TWILIO_PHONE_NUMBER=your_twilio_phone_number
```

4. Run the application

```bash
python app.py
```
